# Public Finance Dashboard

**Objective:** To visually synthesize Indian fiscal data and state-level fiscal indicators for rapid policy analysis.
[cite_start]**Data:** State/Union Territory fiscal datasets[cite: 2, 9].
**Methodology:** Developed a Power BI dashboard focusing on KPI tracking and expenditure trends.
**Outputs:** A `.pbix` source file and a high-resolution `.pdf` export for immediate viewing.

**Key Findings:**
* [cite_start]**Liability Burdens:** The Outstanding Liabilities-to-GSDP ratio [cite: 7] [cite_start]saw a significant pandemic-era spike in 2020-21 [cite: 26][cite_start], breaching 30%[cite: 8]. [cite_start]While it dipped temporarily in the following years, the ratio is trending upward again in 2023-24[cite: 29].
* [cite_start]**Capital Expenditure Push:** The Capital Outlay-to-GSDP ratio [cite: 11] [cite_start]hovered near 2.0% [cite: 30] [cite_start]during 2020-21[cite: 40]. [cite_start]However, it recorded a massive surge in 2023-24 [cite: 43][cite_start], sharply crossing the 3.0% mark[cite: 16]. This points to a heavy recent prioritization of state-level capex despite high liability levels.
